# Flask Integration

## Using the standard client

### option 1: using environment variables

```python
from config import FlaskConfig

from flask import Flask, jsonify


app = Flask(__name__)
FlaskConfig(app)


@app.route('/')
def home():
    return "Hello World!"


@app.route('/consul')
def consul():
    return jsonify(
        consul_port=app.config.get('spring.cloud.consul.port'),
        consul_host=app.config['spring']['cloud']['consul']['host']
    )

@app.route('/config')
def config():
    return jsonify(
        config=app.config
    )
```

### option 2: using custom client

```python
from config import ConfigClient, FlaskConfig

from flask import Flask, jsonify

app = Flask(__name__)
FlaskConfig(app, ConfigClient(app_name='simpleweb000'))

@app.route('/')

    return "Hello World!"

```
